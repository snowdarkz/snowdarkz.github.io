📋 Prompt 
อ่านไฟล์ `bookmory.db` (Sembast NDJSON format) แล้วสร้าง HTML ชั้นหนังสือสำหรับหนังสือที่อ่านจบในปี 2026 โดยมีเงื่อนไขดังนี้:

**หนังสือที่นับ:**
- status = DONE และ cur_page == total_page (อ่านครบทุกหน้า = อ่านจบจริง)

**โทนสีและ Aesthetic:**
- โทนสีโดยรวมเป็น dark warm wood — พื้นหลังสีน้ำตาลเข้มเกือบดำ, ชั้นหนังสือเป็นไม้สีน้ำตาลทอง มี grain texture และ shadow ให้ดูมีมิติ
- ตัวอักษรใช้ฟอนต์ serif สวยงาม (เช่น Playfair Display) สำหรับหัวข้อ และ sans-serif ที่รองรับภาษาไทย (เช่น Sarabun) สำหรับข้อความทั่วไป
- สีหนังสือแต่ละ series ใช้โทนสีเข้ม rich และ saturated (เช่น burgundy, navy, forest green, amber) ไม่ใช้สีพาสเทลหรือสีสว่างจ้า
- แต่ละ series มี gradient บนสันหนังสือ (เข้ม→กลาง→สว่าง→กลาง→เข้ม) เพื่อให้ดูเป็น 3D
- accent color ใช้สีทอง #e8c87a สำหรับชื่อ, stats, และ border ของ tooltip
- tooltip พื้นหลังสีน้ำตาลเข้มแบบ gradient มี border สีทองจางๆ และ drop shadow หนัก

**การแสดงผล:**
- คิดว่าชั้นละ 1 เมตร มีกี่ชั้นก็ได้ตามความเหมาะสม
- ความกว้างสันหนังสือ = ความหนา (คิดจาก 200 หน้า = 1 cm) แปลงเป็น px
- Series เดียวกัน: สีเหมือนกัน + ความสูงเท่ากัน
- ต่าง series: มีความแตกต่างกัน เช่น เอียงเล็กน้อย หรือความสูงต่างกัน
- แสดงวันที่อ่านจบเหนือสันหนังสือแต่ละเล่ม (dd MMM) โดยวันที่เปลี่ยนจะ highlight สีทองชัดขึ้น

**Responsive (มือถือ):**
- ต้องดูได้ดีบนมือถือ ห้าม layout เบี้ยว
- ชั้นหนังสือให้ scroll แนวนอนได้ (overflow-x: auto) แต่หนังสือต้องไม่ wrap ลงบรรทัดใหม่
- ขนาด font, tooltip, และ spacing ต้องปรับให้เหมาะกับหน้าจอเล็ก
- tooltip บนมือถือให้แสดงตรงกลางหน้าจอแทนการติดขอบ

**Hover / Touch Tooltip:**
- tooltip ต้องวางที่ระดับ body เป็น position: fixed คำนวณตำแหน่งด้วย JS ห้ามถูก clip โดย parent element
- รองรับทั้ง mouse (hover) และ touch (tap เพื่อเปิด, tap นอกเพื่อปิด)
- แสดง: รูปปก · ชื่อหนังสือ · ผู้แต่ง · สำนักพิมพ์ · จำนวนหน้า · ความหนา cm · วันที่อ่านจบ
- tooltip ต้องไม่ชนขอบหน้าจอ (clamp ทั้ง X และ Y)
- รูปปกสูง 300px กว้าง tooltip 260px

**Image URL:**
- แปลง URL รูปปกจาก https://m.media-amazon.com/images/S/ → https://i.gr-assets.com/images/S/ เพื่อให้โหลดได้จริง (Goodreads CDN allow hotlink)

**สำคัญ:** ห้ามใส่ overflow: hidden ใน .shelf-container หรือ .books-row เพราะจะทำให้ tooltip ถูกตัด
